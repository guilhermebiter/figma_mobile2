# figma_mobile2
 Aqui estão os prótotipos das telas do app UCB Scout Futebol, que é um aplicativo de estatísticas de futebol que fornece dados em tempo real sobre partidas, jogadores e equipes. Utiliza o padrão MVVM para garantir uma estrutura de código limpa e modular. Ideal para torcedores, analistas e profissionais do esporte.
